# assets/

No logo file, icon set, or illustration asset exists anywhere in the source repo
(`t-cmykh/une.deux-postMaker`) — the repo is pure HTML/CSS/JS editor tooling plus
markdown content templates. There is nothing to copy in here.

Per design-system policy: **no logo/icon was invented to fill this gap.** Everywhere
a mark would go, une·deux renders its name in type — see `guidelines/brand-wordmark.html`
and the `BrandHeader` component (ring monogram "1·2" drawn with a CSS border-circle +
the wordmark "une·deux" set in Anton). If real logo files, an icon font, or illustrations
exist for the brand, add them here and update `BrandHeader`/readme accordingly.
